# 宠物商城移动端 - 技术框架说明

## 项目概述

宠物商城移动端（petstore-mobile）是宠物商城 PC 端（petstore-frontend）的移动适配版本，共用同一套 Spring Boot 后端服务。项目名称为"宠物商城"，版本 1.0.0。

## 技术栈

| 类别 | 技术方案 |
|------|----------|
| **跨平台框架** | uni-app（DCloud） |
| **前端框架** | Vue 3 |
| **开发语言** | JavaScript |
| **构建工具** | Vite |
| **CSS 预处理器** | SCSS |
| **状态管理** | Vue 3 Composition API（reactive） |
| **IDE** | HBuilderX |
| **目标平台** | H5（Web）、Android、iOS、微信/支付宝/百度/头条小程序 |

## uni-app 框架简介

uni-app 是 DCloud 推出的跨平台开发框架，基于 Vue.js 语法，一套代码可以编译到多个平台。本项目主要利用以下特性：

- **条件编译**：通过 `#ifdef` / `#ifndef` 指令针对不同平台编写差异化代码
- **rpx 响应式单位**：750rpx = 屏幕宽度，自动适配不同尺寸设备
- **平台 API 封装**：统一的 `uni.request`、`uni.getStorageSync` 等跨平台 API
- **H5 路由模式**：使用 hash 模式路由，开发端口 3000

## UI 与样式方案

- **无外部 UI 组件库**：全部 UI 均手写实现，使用 scoped SCSS
- **设计系统**：基于 "Chirp Design System" 风格，定义在 `uni.scss` 中
- **主色调**：品牌蓝 `#1c49c2`、品牌橙 `#ff6c10`、价格红 `#bd2848`
- **布局单位**：使用 `rpx`（responsive pixel）实现响应式适配
- **BEM 命名规范**：CSS 类名采用 BEM 风格（如 `.product-card__title`）

## 状态管理方案

采用 Vue 3 Composition API 的 `reactive()` 函数实现轻量级状态管理：

- `stores/user.js` — 管理用户登录状态、Token、用户信息，与 localStorage 双向同步
- `stores/cart.js` — 管理购物车商品列表、数量计算、总价计算，与 localStorage 双向同步

## 数据持久化

使用 `uni.getStorageSync` / `uni.setStorageSync`（底层为 localStorage）存储以下数据：

| 存储 Key | 用途 |
|----------|------|
| `petstore_token` | 用户认证 Token |
| `petstore_user` | 用户信息 |
| `petstore_cart` | 购物车商品列表 |
| `petstore_orders` | 订单列表 |
| `petstore_checkout_items` | 结算页临时数据 |
| `petstore_rememberedUser` | 登录页记住的用户名 |

## 当前数据状态

目前项目**全部使用 Mock 数据**，尚未对接真实后端 API。`api/request.js` 中已封装好 HTTP 请求方法（get/post/put/del），支持自动附加 Bearer Token 和 401 自动跳转登录，等待后端就绪后可直接对接。

## 页面功能一览

| 页面 | 路径 | 功能 |
|------|------|------|
| 首页 | `pages/index/index` | 轮播图、分类入口、热门商品、推荐商品、AI 助手入口 |
| 分类 | `pages/category/category` | 左侧分类栏 + 右侧子分类展示 |
| 商品列表 | `pages/product/list` | 搜索、分类筛选、排序、无限滚动 |
| 商品详情 | `pages/product/detail` | 商品图片、规格选择、评价、加购/立即购买 |
| 购物车 | `pages/cart/cart` | 商品列表、数量增减、删除、结算 |
| 确认订单 | `pages/order/checkout` | 地址选择、商品确认、价格汇总、提交订单 |
| 订单列表 | `pages/order/list` | 按状态筛选（待付款/待发货/待收货/已完成） |
| 订单详情 | `pages/order/detail` | 订单进度、地址、商品、价格明细、操作按钮 |
| 个人中心 | `pages/user/profile` | 用户信息、订单快捷入口、功能菜单 |
| 收货地址 | `pages/user/address` | 地址列表、新增/编辑/删除、省市区选择 |
| 登录 | `pages/auth/login` | 用户名密码登录、记住密码、微信登录（占位） |
| 注册 | `pages/auth/register` | 用户名、手机验证码、密码强度检测、协议确认 |
| 附近店铺 | `pages/shop/nearby` | 店铺搜索、距离/评分展示 |
| 宠物视频 | `pages/video/list` | 视频封面列表、播放按钮 |
| AI 助手 | `pages/ai/chat` | 智能问答、快捷问题、商品推荐（关键词匹配） |

## 后端对接说明

后端基于 Spring Boot，API 基础地址为 `http://localhost:8080/api`。移动端 `manifest.json` 中已配置 H5 开发代理转发。对接时只需将 `api/*.js` 中的 Mock 调用替换为 `request.js` 的 HTTP 请求即可。
