# 后端添加清单 — 第四轮

> 以下均为个人资料编辑功能需要但后端尚未支持的改进。
> 请在 `petstore-server` 中实现，实现后告知前端同事即可对接。

---

## 1. 用户资料更新支持 gender 和 email 字段

**现状：** `UserService.update()` 只处理 `nickname` 和 `avatar`，忽略 `gender` 和 `email`。
**问题：** 前端个人资料页保存性别和邮箱时，请求发了但后端没有存入数据库。

**修改位置：** `UserService.update()` 方法

**改为：**
```java
public User update(User user) {
    User exist = getById(user.getId());
    if (user.getNickname() != null) exist.setNickname(user.getNickname());
    if (user.getAvatar() != null) exist.setAvatar(user.getAvatar());
    if (user.getGender() != null) exist.setGender(user.getGender());
    if (user.getEmail() != null) exist.setEmail(user.getEmail());
    if (user.getPassword() != null && !user.getPassword().isEmpty()) {
        exist.setPassword(user.getPassword());
    }
    return userRepository.save(exist);
}
```

## 2. users 表补充 gender 和 email 字段

**现状：** 如果 `users` 表还没有 `gender` 和 `email` 列，需要先加。

```sql
ALTER TABLE users ADD COLUMN gender VARCHAR(10) DEFAULT 'secret';
ALTER TABLE users ADD COLUMN email VARCHAR(100);
```

`User.java` 实体同步添加：
```java
private String gender = "secret";
```

## 3. Video 实体添加 cover 字段

**现状：** `videos` 表没有封面图字段，手机端视频列表无法显示封面。
**需求：** 支持每个视频关联一张封面图片。

**修改位置：** `Video.java` 实体

```java
private String cover;  // 封面图路径，如 /uploads/covers/xxx.jpg
```

`videos` 表同步加列：
```sql
ALTER TABLE videos ADD COLUMN cover VARCHAR(500);
```

---

## 4. FileService 添加封面上传方法

**修改位置：** `FileService.java`

新增 `uploadCover(MultipartFile file)` 方法，逻辑与 `uploadVideo` 类似：
- 保存目录：`${file.upload-dir}/covers/`
- 返回路径：`/uploads/covers/uuid.jpg`

---

## 5. AdminController 视频创建/编辑支持封面上传

**修改位置：** `AdminController.java`

### POST `/api/admin/videos`（创建）
增加 `MultipartFile coverFile` 参数，有文件时调用 `fileService.uploadCover(coverFile)` 获取封面路径，传入 `videoService.add(...)`。

### PUT `/api/admin/videos/{id}`（编辑）
增加 `MultipartFile coverFile` 参数，支持替换封面图。有新文件时上传并更新 cover 字段。

**修改位置：** `VideoService.java`

`add(...)` 和 `update(...)` 方法签名增加 `String cover` 参数，保存到实体。

## 前端对接说明

后端实现后，前端无需额外接口改动：
- **管理后台** `VideoManage.vue` 会在 FormData 中携带 `coverFile` 字段上传封面图

## 7. Product 实体添加 subcategory 字段

**现状：** `products` 表没有子分类字段，前端宠物周边的细分图标（狗粮、猫砂、鱼缸等）无法真正过滤商品。
**需求：** 支持按子分类筛选商品，如"狗粮"、"猫玩具"、"鱼缸"等。

**修改位置：** `Product.java` 实体

```java
private String subcategory;  // 子分类，如 dog_food, cat_litter, fish_tank
```

`products` 表同步加列：
```sql
ALTER TABLE products ADD COLUMN subcategory VARCHAR(50);
```

## 8. ProductService 查询支持 subcategory 筛选

**修改位置：** `ProductService.java`

查询方法增加 `subcategory` 参数，当传入时按子分类过滤：

```java
public Page<Product> list(String keyword, String category, String productType, 
                          String subcategory, String brand, ...) {
    // 在现有查询条件基础上，增加：
    if (subcategory != null && !subcategory.isEmpty()) {
        criteria.add(cb.equal(root.get("subcategory"), subcategory));
    }
}
```
