# 后端添加清单 — 第四轮

> 以下均为个人资料编辑功能需要但后端尚未支持的改进。
> 请在 `petstore-server` 中实现，实现后告知前端同事即可对接。

---

## 1. 用户资料更新支持 gender 和 email 字段

**现状：** `UserService.update()` 只处理 `nickname` 和 `avatar`，忽略 `gender` 和 `email`。
**问题：** 前端个人资料页保存性别和邮箱时，请求发了但后端没有存入数据库。

**修改位置：** `UserService.update()` 方法

**改为：**
```java
public User update(User user) {
    User exist = getById(user.getId());
    if (user.getNickname() != null) exist.setNickname(user.getNickname());
    if (user.getAvatar() != null) exist.setAvatar(user.getAvatar());
    if (user.getGender() != null) exist.setGender(user.getGender());
    if (user.getEmail() != null) exist.setEmail(user.getEmail());
    if (user.getPassword() != null && !user.getPassword().isEmpty()) {
        exist.setPassword(user.getPassword());
    }
    return userRepository.save(exist);
}
```

## 2. users 表补充 gender 和 email 字段

**现状：** 如果 `users` 表还没有 `gender` 和 `email` 列，需要先加。

```sql
ALTER TABLE users ADD COLUMN gender VARCHAR(10) DEFAULT 'secret';
ALTER TABLE users ADD COLUMN email VARCHAR(100);
```

`User.java` 实体同步添加：
```java
private String gender = "secret";
private String email;
```
